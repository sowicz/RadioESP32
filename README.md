## Radio ESP32 - 32D

##

Sterownik radia z enkoderem i OLED + DAC PCM5102A I2S.
Przetestowano na radiostacji, dzwięk OK.

## Opis

Urządzenie pozwala na sterowanie menu, wybór stacji radiowych i regulację głośności za pomocą enkodera obrotowego z przyciskiem oraz wyświetlacza OLED. Kod jest podzielony na moduły: enkoder, menu, ekrany, lista stacji, wyświetlacz, zegarek.
